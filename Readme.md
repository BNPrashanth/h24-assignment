# H24 Analyser Service

A simple webapp that will accept a url and return the analysis of the HTML content of the page.


## Usage

### Getting started

```bash
$ git clone https://github.com/BNPrashanth/h24-assignment.git
$ cd h24-assignment
```

### Running the app

```bash
$ make go-run
```

### Building the app

```bash
$ make go-build
```

### Cleaning build files

```bash
$ make go-clean
```

### Running the tests

```bash
$ make test
```
